# ⭐️ Установка перевода сборки

В данном небольшом руководстве мы описали инструкции для установки перевода StoneBlock 4 двумя способами!

## Важная информация
- Версия Minecraft для сборки 1.21.1 
- Загрузчик модов NeoForge

## ❗️Способ 1 - Установка с нуля:
1. Скачайте архив с переводом [Google Disk](https://drive.google.com/file/d/1AAD_wq1YdBk4byolbxJ_CyrwgZE6yTKP/view?usp=sharing)
2. Очистите папку .minecraft
3. Перенесите все папки из архива в .minecraft
4. Запустите игру
5. Приятной игры!

## ❗️Способ 2 - Установка в уже имеющуюся сборку:
1. Скачайте архив с переводом [Google Disk]
2. Перенесите папки config, kubejs, resourcepacks с заменой в .minecraft
3. Запустите игру
4. Приятной игры!

## ❗️Скачать оригинальную версию сборки
